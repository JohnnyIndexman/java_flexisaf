# Java Flexisaf Week 1 Project

This project contains a simple Java program that demonstrates the basic structure of a Java application. The program prints "Hello World" to the console.

## Project Structure

The main code resides in the `week1` package and is executed from the `Main` class.

How to Run
Clone the Repository:
git clone https://github.com/JohnnyIndexman/java_flexisaf.git
cd java_flexisaf/week1

Compile the Program: Ensure you have Java installed. Compile the program using:
javac Main.java

Run the Program: After compiling, run the program:
java Main

Expected Output:
Hello World

Requirements
Java Development Kit (JDK): Version 8 or above.
A text editor or an Integrated Development Environment (IDE) such as IntelliJ IDEA, Eclipse, or Visual Studio Code.

Purpose
This project serves as an introductory exercise in:
Understanding the Java syntax.
Writing a simple program to output text.
Structuring a basic Java application with a main method.
